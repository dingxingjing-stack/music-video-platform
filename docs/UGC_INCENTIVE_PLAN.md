### 1. 下载模板 SDK

```
官网 → 创作者中心 → 模板 SDK 下载
包含:
  • 模板结构说明文档
  • 示例模板
  • 提交工具
```

### 2. 创作模板

```
模板结构:
  template.json (配置)
  preview.mp4 (预览视频)
  assets/ (素材文件夹)
  thumbnail.jpg (缩略图)
```

### 3. 提交审核

```
创作者中心 → 提交作品 → 填写信息 → 上传文件
审核周期：3-5 个工作日
```

### 4. 评级发奖

```
审核通过 → 评级 → 公示 3 天 → 发放奖金 → 模板上线
```
