## 🎯 制作标准

### 画质要求
- 分辨率：1080P (1920x1080)
- 帧率：24fps 或 30fps
- 码率：≥8 Mbps
- 格式：MP4 (H.264)

### 音频要求
- 采样率：44.1kHz
- 位深：16-bit
- 响度：-14 LUFS (Spotify 标准)
- 格式：AAC 256kbps

### 剪辑要求
- 转场流畅 (无黑场/跳帧)
- 音画同步 (误差<50ms)
- 字幕无错别字
- 色彩统一 (无偏色)
