### 5. 零成本导出 🎬

**FFmpeg.wasm 流程**:

```javascript
1. 加载 FFmpeg.wasm (CDN)
2. 写入文件到虚拟文件系统
   - FS('writeFile', 'audio.mp3', audioData)
   - FS('writeFile', 'clip_1.mp4', videoData)
3. 执行 FFmpeg 命令
   - concat 所有片段
   - 混合音频
   - H.264 编码
4. 读取输出
   - FS('readFile', 'output.mp4')
5. 创建 Blob URL 下载
```

**导出配置**:
- 分辨率：480p / 720p / 1080p / 4K
- 帧率：24 / 30 / 60 fps
- 格式：MP4 / WebM / GIF
- 质量：低 / 中 / 高 / 超高
