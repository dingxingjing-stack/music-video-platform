## 📊 技术栈

### 后端
- **框架**: FastAPI (Python 3.12)
- **数据库**: SQLite (可扩展 PostgreSQL)
- **AI 集成**: Mureka API, Gemini API
- **WebSocket**: 实时协作/通知
- **文件存储**: 本地/云存储

### 前端
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **状态管理**: Zustand/Context
- **音频处理**: Web Audio API
- **MIDI**: WebMIDI API
- **PWA**: Service Worker + IndexedDB

### 基础设施
- **版本控制**: Git
- **包管理**: npm/pip
- **编译器**: GCC 16.1.0 (VST)
- **框架**: JUCE (VST 开发)
