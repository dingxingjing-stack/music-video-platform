## 📋 四个核心模块

### 1. 交互式引导 (首次打开)
**触发条件**: 新用户首次访问

**流程**:
1. 欢迎动画 (3s)
2. 平台介绍 (5 屏轮播)
3. 年龄验证 (合规要求)
4. 快速创作 (>13 岁跳过教学)

**引导屏内容**:
```
屏 1: "AI 音乐 + MV，3 分钟搞定"
  [示例 MV 播放]
  [下一步]

屏 2: "输入歌词，选择风格"
  [歌词输入演示]
  [下一步]

屏 3: "AI 自动生成专业音质"
  [EQ/压缩可视化]
  [下一步]

屏 4: "一键同步 MV"
  [模板展示]
  [下一步]

屏 5: "开始创作"
  [立即体验] [查看教程]
```

### 2. 功能亮点提示 (渐进式)
**策略**: 用户触发功能时才提示

| 功能 | 触发时机 | 提示内容 |
|------|----------|----------|
| AI 生成 | 首次打开创作页 | "试试输入歌词，AI 帮你作曲" |
| 多轨编辑 | 打开时间轴 | "拖拽素材到轨道，双击编辑" |
| 效果器 | 点击效果器图标 | "5 种效果器，点击试听" |
| 声音克隆 | 打开克隆页 | "上传 1 分钟音频，克隆你的声音" |
| 素材库 | 打开素材页 | "100 个免费素材，任意使用" |
| 社区发布 | 点击发布按钮 | "分享作品，赚取收益" |

**实现方式**:
```tsx
// Tooltip 组件
<Tooltip 
  trigger="first-access"
  content="提示内容"
  position="bottom"
/>

// 引导遮罩
<OnboardingOverlay
  step={currentStep}
  onSkip={() => setSkipped(true)}
/>
```

### 3. 模板视频教程 (内嵌)
**形式**: 3-5 分钟短视频，内嵌于功能页

| 视频 | 位置 | 时长 |
|------|------|------|
| 《3 分钟创作第一首歌》 | 创作页 | 3min |
| 《多轨编辑速成》 | 编辑页 | 4min |
| 《AI 修音黑科技》 | 效果器页 | 4min |
| 《声音克隆教学》 | 克隆页 | 5min |
| 《素材库使用指南》 | 素材页 | 3min |

**实现方式**:
```tsx
// VideoTutorial 组件
<VideoTutorial
  videoId="tutorial-studio-01"
  title="3 分钟创作第一首歌"
  duration="3:00"
  embedUrl="https://cdn.example.com/tutorials/studio-01.mp4"
/>
```
