## 1. 快速开始

### 系统要求
- Node.js 18+
- Python 3.11+
- 现代浏览器 (Chrome/Edge/Firefox)

### 启动应用

```bash
# 终端 1 — 启动后端
cd backend
uvicorn main:app --port 8000

# 终端 2 — 启动前端
cd frontend
npx vite --port 3001
```

### 访问应用
- **主页**: http://localhost:3001
- **API 文档**: http://localhost:8000/docs

## 2. AI 音乐生成

### 功能说明
使用 AI 一键生成完整歌曲，支持人声选择、风格控制、歌词编辑和结构编排。

### 操作步骤

1. **打开主页** → 点击"路径 A — Suno 风格"
2. **输入提示词** (至少 5 个字符)
   ```
   例：一首温暖的中文流行歌曲，关于梦想和希望
   ```
3. **选择人声**: AUTO / 男声 / 女声 / 纯音乐
4. **调整风格滑块**:
   - Weirdness (创意度): 0-100%
   - Style Strength (风格强度): 0-100%
5. **编辑歌词** (可选):
   ```
   [Verse]
   歌词内容...
   
   [Chorus]
   副歌歌词...
   ```
6. **编排结构**: 拖拽段落顺序 (Intro → Verse → Chorus → Bridge → Outro)
7. **点击"✨ 🎵 生成"**

### 注意事项
- 提示词至少 5 个字符
- 开发模式使用 Mock 音频 URL
- 生产模式使用 Mureka API 生成真实音频

## 4. 专业混音台

### 功能说明
专业 MixConsole 混音台，提供通道条、效果器、Aux 发送等完整混音功能。

### 操作步骤

1. **打开混音台**: 多轨编辑器 → 点击"🎚️ 混音台"
2. **调整音量**: 拖动通道推子
3. **声像控制**: 左右拖动声像旋钮
4. **静音/独奏**: 点击 Mute/Solo 按钮
5. **效果器**: 在效果器插槽中选择效果
6. **Aux 发送**: 调整发送量

### 支持的效果器
- Compressor (压缩器)
- Reverb (混响)
- EQ (均衡器)
- Delay (延迟)
- Chorus (合唱)

## 5. 分轨导出

### 功能说明
AI 智能分轨导出，将音频分离为人声、鼓、贝斯等独立音轨。

### 操作步骤

1. **点击"📤 导出音频"**
2. **勾选"AI Smart Stems"**
3. **选择分轨类型**:
   - 🎤 Vocals (人声)
   - 🥁 Drums (鼓)
   - 🎸 Bass (贝斯)
   - 🎵 Other (其他)
4. **选择格式**: WAV / MP3
5. **点击"导出"**

### API 调用
```bash
POST /api/v1/export/stems
Content-Type: application/json

{
  "audio_url": "音频URL",
  "stems": ["vocals", "drums", "bass", "other"]
}
```

## 6. 音阶辅助

### 功能说明
Scale Assistant 帮助用户在 MIDI 编辑时选择正确音符，防止弹错音。

### 支持的音阶 (8 种)
| 音阶 | 说明 |
|------|------|
| Major | 大调 (欢快明亮) |
| Natural Minor | 自然小调 (忧伤) |
| Harmonic Minor | 和声小调 |
| Melodic Minor | 旋律小调 |
| Major Pentatonic | 大调五声 (中国风) |
| Minor Pentatonic | 小调五声 (蓝调) |
| Blues | 蓝调 |
| Chromatic | 半音阶 (全部) |

### 操作步骤

1. **打开 MIDI 编辑器**
2. **选择根音**: C, C#, D... (12 个)
3. **选择音阶类型**
4. **键盘高亮**: 音阶内音符会高亮显示
5. **自动修正**: 开启后弹错音自动修正

## 7. 音高修正

### 功能说明
检测音频音高并量化到目标音阶，类似 Cubase VariAudio 的简化版。

### 操作步骤

1. **打开音高修正面板**
2. **选择根音和音阶**
3. **点击"分析音高"**: 等待分析完成
4. **调整修正强度**: 0-100%
5. **点击"自动修正"**

### API 调用
```bash
GET /api/v1/pitch/scales  # 获取音阶列表
POST /api/v1/pitch/correct  # 执行修正
```

## 8. 和弦轨道

### 功能说明
内置 26 个和弦库，支持自动和弦进行生成和和声编排。

### 和弦库

| 类型 | 和弦 |
|------|------|
| 大三和弦 | C D E F G A B |
| 小三和弦 | Cm Dm Em Fm Gm Am Bm |
| 七和弦 | C7 D7 E7 F7 G7 A7 B7 |
| 减和弦 | Cdim Ddim |
| 增和弦 | Caug |

### 常用和弦进行 (6 种)
| 进行 | 和弦 | 风格 |
|------|------|------|
| I-V-vi-IV | C-G-Am-F | 流行 |
| ii-V-I | Dm-G-C | 爵士 |
| I-IV-V | C-F-G | 蓝调 |
| vi-IV-I-V | Am-F-C-G | 流行 |
| I-vi-IV-V | C-Am-F-G | 50s |
| i-VII-VI-V | Am-G-F-E | 小调 |

### 操作步骤

1. **打开和弦轨道面板** (3 个 Tab)
2. **Tab 1 — 和弦库**: 浏览 26 个和弦
3. **Tab 2 — 和弦进行**: 选择模板，点击"生成"
4. **Tab 3 — 和声编排**: 选择编排方式 (柱式/分解/长音)

## 9. Comping 多次录制

### 功能说明
录制多次后选择最佳片段拼接，支持评分和可视化时间线。

### 操作步骤

1. **打开 Comping 面板**
2. **设置录音次数**: 2-20 次
3. **点击"创建 Comping 会话"**
4. **评分**: 点击星星 (1-5 星)
5. **选择片段**: 点击录音轨道卡片
6. **编译**: 点击"✨ 编译最佳片段"
7. **导出**: 编译成功后导出

## 10. 时间伸缩

### 功能说明
变速不变调，支持 BPM 检测、Warp Marker 和节拍量化。

### 操作步骤

1. **打开时间伸缩面板**
2. **检测 BPM**: 点击"🎵 检测 BPM"
3. **设置目标 BPM**: 输入目标值
4. **查看伸缩比例**: 自动计算
5. **Warp Marker**: 
   - 锁定/解锁标记
   - 点击"锁定"按钮
6. **量化**: 选择网格分辨率 (16 分/8 分/4 分)
7. **应用**: 点击"🚀 应用伸缩"

## 11. Remix 引擎

### 功能说明
AI 风格转换，10 种 Remix 风格，一键重新编曲。

### 支持的风格 (10 种)

| 风格 | BPM 范围 | 特点 |
|------|----------|------|
| Electronic | 120-150 | 四拍子低音 |
| Lofi | 70-90 | 慵懒黑胶质感 |
| Ambient | 60-80 | 空灵氛围 |
| Rock | 100-140 | 失真吉他 |
| Jazz | 80-120 | Swing 节奏 |
| Cinematic | 60-100 | 史诗管弦 |
| Trap | 130-160 | 808 低音 |
| House | 120-130 | 经典电子 |
| Dubstep | 140-150 | Wobble Bass |
| Acoustic | 80-120 | 不插电 |

### 操作步骤

1. **打开 Remix 面板**
2. **选择目标风格** (10 选 1)
3. **选择强度**: 
   - 轻微 (20-40%)
   - 中等 (40-70%)
   - 极端 (70-100%)
4. **高级选项**:
   - 节奏倍率: 0.5x - 2.0x
   - 添加 Drop: 勾选
   - 添加 Buildup: 勾选
5. **点击"✨ 开始 Remix"**
6. **查看结果**: 显示应用的变化列表
7. **导出**: 点击"📥 导出 Remix"

## 12. 声音克隆

### 功能说明
上传 1-5 分钟声音样本，AI 学习后模仿合成。

### 录制要求
- ✅ 清晰人声，无背景噪音
- ✅ 时长 1-5 分钟
- ✅ 格式：WAV 或 MP3
- ✅ 采样率 ≥ 44.1kHz
- ✅ 单一说话人

### 操作步骤

1. **打开声音克隆面板** (3 个 Tab)
2. **Tab 1 — 上传样本**:
   - 点击"📁 选择音频文件"
   - 等待上传完成
3. **Tab 2 — 声音克隆**:
   - 选择声音
   - 输入合成文本 (最多 1000 字符)
   - 调整速度 (0.5x - 2.0x)
   - 点击"🎙️ 开始克隆"
4. **Tab 3 — 音色库**:
   - 浏览已上传的声音
   - 点击"使用此声音"

## 14. 常见问题

### Q: 启动报错怎么办？
**A**: 
1. 检查端口是否被占用: `netstat -ano | grep ":8000"`
2. 杀掉占用进程: `taskkill /PID <PID> /F`
3. 重新启动服务

### Q: AI 生成返回 404？
**A**: 确保后端路由有 `prefix="/api/v1/ai"`。已在 v2.0 修复。

### Q: 前端页面空白？
**A**: 按 `Ctrl+F5` 硬刷新浏览器缓存。

### Q: API 如何查看？
**A**: 访问 `http://localhost:8000/docs` 查看 Swagger 文档。

### Q: 如何切换 AI 后端？
**A**: 修改 `backend/main.py`:
```python
# 开发模式 (Gemini 免费额度)
from app.routers import gemini_ai_music as ai_music

# 生产模式 (Mureka API)
from app.routers import ai_music
```

### Q: 支持哪些音频格式？
**A**: WAV, MP3, FLAC (导出)。上传支持 WAV, MP3。
