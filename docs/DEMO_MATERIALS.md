## 🎬 录制操作顺序

### 第一部分：开场 (30秒)
1. 打开 http://localhost:3001
2. 展示主页 4 个路径卡片
3. 点击"多轨编辑"

### 第二部分：AI 生成 (60秒)
1. 点击"✨ AI 生成"
2. 选择人声：**女声**
3. 拖动 Weirdness：**60%**
4. 拖动 Style Strength：**80%**
5. 粘贴歌词示例
6. 点击"生成"

### 第三部分：混音台 (30秒)
1. 点击"🎚️ 混音台"
2. 拖动主推子从 80% → 100%
3. 点击某个轨道的 Mute
4. 关闭混音台

### 第四部分：P1 功能快速展示 (90秒)
1. **Scale Assistant**: 选择 C / Major
2. **音高修正**: 点击分析 (跳过等待)
3. **Chord Track**: 显示和弦库
4. **Comping**: 创建会话 → 编译
5. **Remix**: 选择 Electronic → 开始
6. **时间伸缩**: 检测 BPM → 伸缩
7. **声音克隆**: 上传 → 合成

### 第五部分：社区 + 结尾 (60秒)
1. 导航 → 社区排行榜
2. 点击 3 个榜单
3. 总结文字
4. 淡出
